# ITSM Leadership Portfolio

This portfolio contains three core work products built to demonstrate structured, practical approaches to IT Service Management and IT Asset Management. Each project provides a clear, repeatable model for monitoring performance, managing assets, and standardizing support operations.

---

## Projects Included

### 1. KPI Dashboard – IT Service Desk
A complete 30-day KPI dataset and dashboard with daily metrics, scoring logic, trend visuals, and a structured action plan. Includes a 1-page summary and a presentation deck with a 90-day operational plan.

➡️ See folder: `/KPI-Dashboard`

---

### 2. IT Asset Lifecycle Case Study
A concise overview of the full hardware lifecycle model, showing how structured processes improve return rates, reduce downtime, and strengthen inventory accuracy.

➡️ See folder: `/Asset-Lifecycle-Case-Study`

---

### 3. Service Desk SOP Toolkit
A foundational SOP package covering ticket handling guidelines, escalation paths, onboarding/offboarding steps, and operational routines for maintaining consistent Service Desk performance.

➡️ See folder: `/Service-Desk-SOP-Toolkit`

---

## Purpose

This portfolio offers examples of well-structured operational documentation, performance monitoring tools, and lifecycle models. It reflects a practical approach to improving service quality, operational predictability, and long-term maintainability within IT environments.
