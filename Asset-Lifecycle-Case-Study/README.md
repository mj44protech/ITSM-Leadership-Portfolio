# IT Asset Lifecycle Case Study

This folder contains a concise case study that illustrates how I would manage the end-to-end hardware lifecycle in a small to mid-sized environment.

## Included Artifact

- `IT_Asset_Lifecycle_Case_Study.pdf`  
  - Describes the lifecycle from procurement to disposal/return.
  - Highlights before/after KPIs:
    - Asset return rate
    - Refresh downtime
    - Inventory accuracy
  - Summarizes ROI and operational benefits.

## How to Use This

In conversations, position this as proof that you understand ITAM + ITSM integration, governance, and the impact of process on cost and risk.
